Αυτό το αποθετήριο περιέχει τον πηγαίο κώδικα για τη συγκριτική ανάλυση και αξιολόγηση μοντέλων Ανακάλυψης Διεργασιών (Process Mining). Η έρευνα επικεντρώνεται στη σύγκριση των κλασικών Directly-Follows Graphs (DFG) με δομές Prefix Trees (Tries) και Υβριδικών Κατευθυνόμενων Ακυκλικών Γραφημάτων (Hybrid DAGs), χρησιμοποιώντας τα πραγματικά event logs BPI Challenge 2017 και 2019.

* Σκοπός του Project
Η μελέτη αξιολογεί τα μοντέλα με βάση τις 4 ποιοτικές διαστάσεις του Process Mining:

Fitness (Καταλληλότητα): Πόσο καλά αναπαράγει το μοντέλο τις συμπεριφορές του log.

Precision (Ακρίβεια): Πόσο περιορίζει το μοντέλο την ανεπίτρεπτη «υπερ-γενίκευση».

Simplicity (Απλότητα): Ο αριθμός των κόμβων και των ακμών (πολυπλοκότητα γραφήματος).

Generalization (Γενίκευση): Η ικανότητα πρόβλεψης μελλοντικών/αόρατων ιχνών μέσω K-Fold Cross Validation.

* Δομή Αρχείων Κώδικα
compare_DFG-Trie_Metrics.py: Υπολογίζει και συγκρίνει τον συνολικό αριθμό κόμβων και ακμών (Simplicity) μεταξύ ενός παραδοσιακού DFG και ενός Pure Trie.

createDFG-Trie.py: Φιλτράρει το log στις κορυφαίες 20 διαδρομές (top-20 variants), ανακαλύπτει το DFG, κατασκευάζει το Prefix Tree και υπολογίζει τις μετρικές Alignment-based Fitness και Precision.

createDAG.py: Υλοποιεί έναν custom αλγόριθμο που μετατρέπει το Prefix Tree σε ένα Βελτιστοποιημένο Υβριδικό DAG (Optimized Tree), εφαρμόζοντας κλάδεμα σπάνιων διαδρομών (Pruning) και σύμπτυξη κοινών καταλήξεων (Suffix Merging).

sensitivity_analysis.py: Πραγματοποιεί Ανάλυση Ευαισθησίας (Sensitivity Analysis). Ελέγχει 5 διαφορετικές τιμές του κατωφλίου κλαδέματος (min_support), μετατρέπει το DAG σε Petri Net και παράγει ένα γράφημα (sensitivity_analysis_plot.png) που δείχνει πώς επηρεάζονται τα Fitness και Precision.

k-fold_validation.py: Εκτελεί 5-Fold Cross Validation για να μετρήσει με επιστημονική ακρίβεια το Test Fitness (Generalization) των δύο προσεγγίσεων σε άγνωστα δεδομένα.

* Προαπαιτούμενα & Εγκατάσταση
Για να τρέξεις τα scripts, θα χρειαστείς Python 3.x και τις παρακάτω βιβλιοθήκες. Μπορείς να τις εγκαταστήσεις εκτελώντας:

pip install pm4py pandas scikit-learn matplotlib graphviz

Σημείωση: Για την οπτικοποίηση των γραφημάτων απαιτείται η εγκατάσταση του εργαλείου Graphviz στο σύστημά σου.
Επίσης, θα πρέπει να τοποθετήσεις τα αρχεία δεδομένων BPI_Challenge_2017.xes.gz και BPI_Challenge_2019.xes.gz στον ίδιο φάκελο με τα αρχεία κώδικα.
